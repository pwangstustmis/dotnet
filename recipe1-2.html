本單元將說明 網頁檔案基本架構、常用HTML標籤、表單HTML標籤

網頁檔案基本架構

HTML為網頁標註語言，設計者利用各種HTML標籤建構網頁，要設計網頁，設計者必須了解各種HTML標籤(tag)及其對應屬性設定，本單元將介紹常用HTML標籤及其屬性，更多HTML標籤介紹，請參考Mozilla HTML文件或W3Schools HTML文件。

網頁設計是將顯示內容置於HTML標籤中，HTML標籤為 <標籤名稱>，標籤名稱一般為英文縮寫，例如標題列英文為(heading)，因此HTML的標題標籤使用 <h1> 代表大標題，其中 h 代表標題列，數字 1 代表大標題，HTML預設有六種標題標籤分別為 h1 到 h6。

大部分網頁設計會將顯示內容置於HTML開始標籤 <標籤名稱> 與結束標籤 </標籤名稱>之間，也有少數標籤為單一自我結束標籤 <標籤名稱/>，下列程式片段為網頁基本架構檔案，網頁檔案均包含於網頁標籤 <html> 與 </html> 之間，標籤<head> 區塊用於網頁宣告，其中包含詮釋資料中字元集設定(charset)、瀏覽器網頁抬頭設定(title)、JavaScript設定與樣式CSS設定等，網頁顯示內容則置於 body 區塊。

HTML標籤除了標籤名稱之外，最重要的是標籤屬性(attribute)設定，HTML利用標籤指定內容所屬的標籤類別，之後利用標籤屬性來設定內容的細項，標籤屬性包含於標籤中如 <標籤名稱 屬性名稱="屬性值">，介紹HTML標籤格式後，接下來說明常用HTML標籤。

<html> <head> <!-- 設定utf-8編碼 --> <meta charset="utf-8"> <title>瀏覽器網頁抬頭</title> <script> <!-- 內嵌JavaScript --> </script> <!-- 連接外部JavaScript檔案 --> <script src='JavaScript檔案位置'></script> <style> <!-- 內嵌CSS樣式 --> </style> <!-- 連接外部CSS樣式檔 --> <link rel='stylesheet' href='外部樣式檔位置'> </head> <body> <!-- 網頁內容 --> </body> </html>
練習：請在ASP.Net Core Web MVC專案中 wwwroot 檔案夾中新增檔名 homepage.html 網頁檔，套用上述原始碼，同時瀏覽器網頁標題為 基礎HTML練習，網頁內容顯示 南臺資管。

常用HTML標籤

常用HTML標籤包含：

  • 標題h(heading)

    • 標籤h預設用於設定標題，有六種預設大小(h1到h6)，每個標題標籤預設會佔一行，可以搭配CSS改變標題樣式。

    練習：請在專案中 wwwroot 檔案夾中 homepage.html 網頁檔內容顯示六種標題(h1到h6)的 南臺資管。
  • 段落p(paragraph)

    • 標籤p用於設定段落內容，每個段落有預設間距，可以搭配CSS改變段落樣式。段落內跳行請使用分行<br/>標籤。

    練習：請在專案中 wwwroot 檔案夾中 homepage.html 網頁檔內容顯示兩個段落，第一個段落包含「abcde」，第二個段落包含「abcde」，但是第二段第一行僅包含「ab」、第二行則包含「cde」。
  • 標註(mark)

    • 標註(mark)用於強調某些內容。

    練習：請在專案中 wwwroot 檔案夾中 homepage.html 網頁檔內容顯示段落內容「abcde」，並利用標註(mark)強調「bc」。
  • 上標(sup)、下標(sub)

    • 用於顯示類似數學符號上標(sup)與下標(sub)。

    練習：請在專案中 wwwroot 檔案夾中 homepage.html 網頁檔，利用上下標顯示 (x1+x2)2。
  • 圖片(img)

    • 標籤img用於顯示圖片，常用屬性為 src 與 alt，src 用於設定圖片檔案位置，如果圖片檔位置在本機，則使用 相對位址 來設定圖片位置，請在 wwwroot 目錄下新增一目錄名稱為 imgs，並在目錄下放置一圖片檔，網頁檔 homepage.html 顯示該圖片的 src設定為 imgs/圖片檔名，如果圖片在遠端，則 src設定為 圖片網址。alt用於設定顯示文字，img標籤可使用 width(寬) 或 height(高) 來設定圖片大小，寬與高值可為絕對大小(實際大小或像素點)或相對大小(%)，通常建議使用樣式來進行控制。

    練習：請在南臺首頁選擇任一照片，並在專案中 wwwroot 檔案夾中 homepage.html 網頁檔，利用img標籤並使用本地端及遠端方式顯示該照片，之後透過 寬與高(絕對與相對) 設定改變圖片顯示大小。
  • 超連結(a)

    • 標籤 a 用於設定超連結，常用屬性包含 href 與target，href 用於設定連結對象，如果連結對象為網站內的資源，請使用相對檔案位置，如果連結對象為網站外部資源，請使用絕對位置(必須指定全部網址)，href 值如果設為 # 代表空連結。target 用於設定連結內容顯示頁面，當設定為 _blank 時，瀏覽器會開啟新的頁面顯示連結內容，如不指定，則連結內容會顯示在現有的頁面。

    練習：請在南臺首頁選擇任一照片，並在專案中 wwwroot 檔案夾中 homepage.html 網頁檔，利用a標籤顯示本地端圖片，利用另一a標籤顯示遠端圖片，搭配 target 屬性讓圖片顯示在現在瀏覽器頁面或是新瀏覽器頁面。
  • 表格(table)

    • table標籤用於設定表格，表格包含列與欄，tr 標籤用於新增一列，如果表格有 5 列，就會有5個tr 開始與結束標籤。
    表格每一欄使用 th 與 td 標籤，th 標籤通常用於設定表格標題，td 標籤用於設定一般儲存格，表格也提供儲存格合併功能，在 th 與 td 標籤中設定 rowspan 與 colspan 屬性來設定合併列或合併欄，下列程式片段宣告一個2列2欄表格、將2列2欄表格中第一欄的1、2列合併、將2列2欄表格中第一列的1、2欄合併。請注意：表格標籤中 border 用於設定邊框粗線，一般建議以CSS樣式設定。

    <table border='1' > <tr> <td> 列1欄1 </td> <td> 列1欄2 </td> </tr> <tr> <td> 列2欄1 </td> <td> 列2欄2 </td> </tr> </table> <!-- 將2列2欄表格中第一欄的1、2列合併 --> <table border='1'> <tr> <td rowspan=2> 列1欄1 </td> <td> 列1欄2 </td> </tr> <tr> <td> 列2欄2 </td> </tr> </table> <!-- 將2列2欄表格中第一列的1、2欄合併 --> <table border='1'> <tr> <td colpan=2> 列1欄1 </td> </tr> <tr> <td> 列2欄1 </td> <td> 列2欄2 </td> </tr> </table>
    練習：請利用表格及圖片設計出麥當勞超值全餐。
  • 項目(ul)、符號(ol)、項目或符號項次(li)

    • 標籤 ul 用於設定無編號項目，ul 標籤中每個項目使用 li 標籤，可搭配樣式設定不同項目樣式。標籤 ol 用於設定編號項目，ol 標籤中每個項目使用 li 標籤，可搭配樣式設定不同編號樣式。

    <ul> <li>項目1</li> <li>項目2</li> <li>項目3</li> </ul> <ol> <li>項目1</li> <li>項目2</li> <li>項目3</li> </ol>
    練習：請利用項目及編號顯示商管學院所有系所。
  • 方塊區域(div、span)

    • div 與 span 標籤用在網頁設定方形區域，這些方形區域可用於放置內容如圖片、文字等，功能有一點像日常生活使用的便利貼，請注意：div 區塊預設會跳行，也就是兩個 div 區域，預設不會在同一列，但是可透過 CSS 樣式來溪改，span預設在同一行，例如在同一行中要包含不同的內容樣式，可使用span。

  • 聲音(audio)

    • audio 標籤用於撥放聲音檔，支援格式包含MP3、WAV與OGG，請注意Apple Safari瀏覽器不支援OGG格式，audio標籤常用屬性包含 controls(用於設定顯示控制按鈕)、autoplay(用於設定自動撥放)、muted(靜音)，audio標籤會包含source標籤用於設定聲音檔來源(src)與格式(type)，請注意：audio 標籤可以設定多種格式，audio會對音源檔由上而下檢查是否可以撥放。

    <audio controls autoplay muted> <source src="聲音檔1位置" type="audio/ogg"> <source src="聲音檔2位置" type="audio/mp3"> 瀏覽器不支援音樂撥放 </audio>
  • 影片(video)

    • video 標籤用於撥放影片檔，支援格式包含MP4、WebM與OGG，請注意Apple Safari瀏覽器不支援OGG格式，video標籤常用屬性包含 controls(用於設定顯示控制按鈕)、autoplay(用於設定自動撥放)、muted(靜音)、width(撥放器寬)、height(撥放器高)、poster(載入時顯示圖片檔)，video 標籤會包含source標籤用於設定影片檔來源(src)與格式(type)，請注意：可以設定多種格式，video會對影片檔由上而下檢查是否可以撥放。

    <video controls muted> <source src="影片檔1位置" type="video/ogg"> <source src="影片檔2位置" type="video/mp4"> 瀏覽器不支援影片撥放 </video>

表單HTML標籤

當使用者想要傳送資料至後端伺服器，表單是最常用的方式，表單控制項標籤需包含於form標籤內，form 標籤常用屬性包含action、method與enctype，action 屬性用於指定伺服器端表單資料處理程式，如未指定action則表單將傳送給自己，method 屬性一般使用get與post值，當method 屬性設為 get 時，表單資料會透過網址列變數名稱與值的方式進行傳送，傳送長度約3000字大小，當method 屬性設為 post時，表單資料會儲存於HTTP request內容中進行傳送，沒有長度限制。enctype屬性限用於method屬性值為post，用於設定傳送表單時表單資料編碼方式，一般為 application/x-www-form-urlencoded 為預設編碼方式，資料傳送會先進行編碼，空白用+取代，其他字元轉為16進位ASCII字元，multipart/form-data 用於上載檔案，傳送方式必須使用post。

<form method="get" action="後端程式位置"> </form> <form method="post" action="後端程式位置"> </form>

常用表單控制項包含 輸入(input)、文字區域(textarea)、下拉式選單(select)、fieldset(群組包欄位) 等，以下針對常用表單控制項進行說明。

  • 輸入(input)

    • 表單輸入標籤 input 是表單中使用最為頻繁的標籤，用於資料輸入，常用屬性包含 type、value、name、placeholder， type 屬性用於設定不同輸入類型包含： text(文字)、button(按鈕)、checkbox(核取方塊)、color(顏色選取)、date(日期選取)、datetime-local(日期及時間選擇)、email(電子郵件輸入)、file(檔案選擇)、hidden(隱藏表單欄位)、image(圖片按鈕)、month(月份選擇)、number(數字選擇器)、password(密碼輸入)、radio(選項按鈕)、range(範圍選擇器)、submit(送出按鈕)、tel(電話號碼)、time(時間選擇器)、week(月份選擇器)。submit(送出按鈕) 類型用於傳送表單，預設以按鈕方式顯示，按鈕文字請設定於 value屬性，詳細屬性設定請參考MDN Wen Docs Input element，如果input 標籤為必須輸入控制項，則input 標籤須包含屬性 required ，要將控制項資料傳送給後端程式，欄位控制項一定要設定 name 屬性，用於後端程式取得對應欄位，input 標籤一般會搭配 label 標籤用於設定該欄位標籤文字，要使用label 標籤，控制項請設定 id 屬性值，label 標籤使用 for 屬性指定該名稱控制項。

    <form> <label for="f1">文字方塊f1名稱</label> <input type="text" id="f1"><p></p> <input type="submit" value="送出"> </form>
    練習：請利用上述原始碼練習各種不同輸入(input)標籤類型包含： text(文字)、checkbox(核取方塊)、date(日期選取)、email(電子郵件輸入)、file(檔案選擇)、hidden(隱藏表單欄位)、password(密碼輸入)、radio(選項按鈕)、range(範圍選擇器)、submit(送出按鈕)。
  • 文字區域(textarea)

    • textarea標籤用於設定多行文字輸入，屬性包含 rows(指定列數) 與 cols(指定每列字數)。

    <form> <textarea cols="40" rows="5"></textarea> </form>
  • 下拉式方塊(select)

    • select標籤用於設定下拉式方塊選項，每個選項使用option標籤。

    <form> <label for="e1">教育程度</label> <select id="e1"> <option value="1">高中(含)以下</option> <option value="2">大專</option> <option value="3">研究所以上</option> </select> </form>
  • 群組表單欄位(fieldset)

    • fieldset標籤用於群組標單欄位，使用 legend標籤設定每個群組標題。

    <form> <fieldset> <legend>教育選項</legend> <label for="e1">教育程度</label> <select id="e1"> <option value="1">高中(含)以下</option> <option value="2">大專</option> <option value="3">研究所以上</option> </select> </fieldset> <fieldset> <legend>收入選項</legend> <label for="e2">收入範圍</label> <select id="e2"> <option value="1">40,000</option> <option value="2">40,001~60,000</option> <option value="3">60,001以上</option> </select> </fieldset> </form>
