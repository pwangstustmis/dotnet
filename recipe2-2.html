Recipe 2：Razor Script介紹

本單元將說明 Razor Script語法、Razor Script條件、Razor Script迴圈

Razor Script語法

Razor是將.NET程式碼內嵌到網頁中的標記語法，Razor語法包含Razor標記、C#和 HTML。伺服器端會解析Razor並將結果呈現於檢視頁面中，透過Razor，設計者設計動態資料導向的網頁應用。在檢視畫面中，設計者可以使用 @ 符號加入Razor標記、C#和 HTML，請注意 @ 後命令不可以有空白。使用Razor語法有下列三種方式：

  • 隱含：直接在 @符號之後輸入Razor運算式，例如在Index檢視頁面(Index.cshtml)中輸入 @DateTime.Now.ToString("yyyy-MMM-dd") 會顯示現在時間的年月日。但是隱含方式的方式僅能用於變數、保留字等，不能用於運算如 3+5 或包含空格。
  • 明確：直接在 @符號之後將Razor運算式放在小括符中，例如在Index檢視頁面(Index.cshtml)中輸入 @(3+5) 會顯示計算結果。明確方式可用於運算如 3+5 或包含空格。
  • 多行：將Razor運算式置於 @{ } 中。在多行命令中可以包含HTML標籤，如 @{ <h1>Hello!</h1> }，同時在HTML標籤內如要使用Razor變數，則需使用 @ 符號，例如 @{ var x="STUST"; <h1>@x</h1> }。

練習：請在HomeController中Index檢視頁面新增變數deposit(本金)值為 10000，變數interest(利率)值為 0.02，並使用Razor多行方式用大標題顯示deposit與interest。

Razor Script條件

Razor語法可使用條件結構 @if、else if、else與 @switch。

@if { } else { } @if { } else if { } else { } @switch (變數) { case 值: break; case 值: break; default: }
練習：請在HomeController中Index檢視頁面設定變數dept值為 資訊管理系，並在對應檢視畫面判斷值是否為 資訊管理系，如果是請印出 是資訊管理系，否則請印出 不是資訊管理系。

Razor Script迴圈

Razor語法可使用迴圈結構 @for、@foreach、@while與@do while。

@for (var 變數=起始值;結束條件;間距值) { } @foreach (var 變數 in 清單) { } @while (條件) { } @do { } while (條件);
練習：請在HomeController中Index檢視頁面設定變數num值為 10，並利用迴圈顯示所有 1 到 變數dept值。
